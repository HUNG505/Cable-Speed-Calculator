# 電纜線速度與轉速計算機 
這是一個專為電纜製程與機械設計人員打造的網頁應用程式，全面優化自適應排版，完美支援電腦與行動裝置（手機巡廠）操作。供製程工程師與現場作業人員快速換算鐵軸直徑、每分鐘轉速與線速度，並掌握收放線過程中的速度動態變化。

## 功能特色 

- **雙向計算模式**：
  - i. **求線速度**：輸入轉速 (RPM)，自動計算出鐵軸外徑 $D$ 與軸底徑 $d$ 對應的切線線速度 (m/min)。
  - ii. **求轉速**：輸入目標線速度 (m/min)，自動反推機台所需的每分鐘轉速 (RPM)，並同步呈現軸底徑之動態速度。
- **動態機台示意圖**：內建動態旋轉 SVG 視覺化模型，收放線旋轉速度依實際 RPM 等比例呈現（經 1/10 視覺縮放以便於現場肉眼觀察），即時標註當前直徑、轉速與線速度關係。
- **即時趨勢關係圖表**：整合 Recharts 互動式線性圖表，在固定轉速下，自動繪製從「軸底徑 $d$」到「鐵軸外徑 $D$」的線速度連續變化曲線，精確掌握滿軸與空軸時的速度落差。
- **行動端響應式與防呆優化**：
  - o 擴展圖表下邊距與軸線偏移，確保手機直向瀏覽時刻度與標籤清晰可見、不被裁切。
  - o 自動連動防呆：變更鐵軸外徑時，軸底徑自動以標準 1/2 比例同步更新（亦支援手動自訂）。
